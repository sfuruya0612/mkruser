# mkruser
CLI to get or remove mackerel users.

## Usage

```bash
$ mkruser -h
NAME:
   mkruser - CLI to get or remove mackerel users.

USAGE:
   mkruser [global options] command [command options] [arguments...]

VERSION:
   20220110-162559 f270c70 (Build by: go version go1.17.5 darwin/amd64)

COMMANDS:
   list, l      Get user list.
   delete, req  Delete user.
   help, h      Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --key value, -k value   [$MACKEREL_API_KEY]
   --help, -h             show help (default: false)
   --version, -v          print the version (default: false)
```
